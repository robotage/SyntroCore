SyntroCore

SyntroCore is the fundamental building block of SyntroNet. It consists of:

  • Libraries:
    • SyntroLib - a library used by all Syntro apps.
    • SyntroGUI - a library that provides common dialogues for windowed SyntroNet apps.
    • SyntroControlLib - a library that allows apps to embed SyntroControl functionality.
  • Syntro apps:
    • SyntroExec - an app that starts and manages other SyntroNet apps.
    • SyntroControl - Syntro's distirbuted message switch and directory manager.
    • SyntroLog - an app to collect and display log entries from all apps in a SyntroNet cloud.
    • SyntroDB - an app that can collect and store multicast service streams from a SyntroNet cloud and make them available via the CFS service.
